Dynasty GM Front Office

How to use:
1. Open dynasty_gm_front_office.html on your phone or computer.
2. Import the HarryKnowsBall CSV from the Import tab.
3. Paste your roster block to mark rostered players.
4. Add/update scouting scores as we evaluate players.
5. Use Backup to export JSON and send it back to ChatGPT.

Important:
Data is stored in the browser's local storage. Export JSON regularly.
